## 📖 Project Overview

The **Quiz Application with Timer** is a console-based Java application developed as part of the **CodSoft Java Programming Internship**.

The application provides an interactive quiz experience where users answer multiple-choice questions within a limited time. It validates user input, calculates scores automatically, displays performance statistics, and follows Object-Oriented Programming (OOP) principles to ensure clean and maintainable code.

This project demonstrates Java fundamentals including classes, objects, collections, loops, conditional statements, multithreading, and user interaction through the console.

✅ User Registration

# 🚀 How to Run

### Clone Repository

```bash
git clone https://github.com/mohammadzunaid333-hash/CodSoft-Quiz-Application-With-Timer.git
```

### Open Project

Open the project in:

- VS Code
- IntelliJ IDEA
- Eclipse

### Compile

```bash
javac *.java
```

### Run

```bash
java Main
```
